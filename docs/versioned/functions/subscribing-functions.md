### Deployment with Triggers

When invoking `func deploy` the CLI will create Knative Triggers for the function.

=== "func"

    Deploy the function with Triggers by running the command inside the project directory:

    ```bash
    func deploy
    ```

=== "kn func"

    Deploy the function with Triggers by running the command inside the project directory:

    ```bash
    kn func deploy
    ```
